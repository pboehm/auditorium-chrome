# Auditorium Chrome Extension

Chrome Extension that asks the [Auditorium](https://github.com/pboehm/auditorium) for new posts and comments and displays them as a badge.

Requires an auditorium account with an authorization token.

## Installation

Download the source code and extract it to a directory. Open Chrome and visit `chrome://extensions/`. Enable developer options and add the directory as an extension.
